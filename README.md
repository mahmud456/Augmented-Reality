# Augmented-Reality
Developed a web-based prototype to improve awareness and accessibility of Augmented Reality (AR) in medical education for Bangladeshi students and healthcare professionals. The platform includes case studies, FAQ support, interactive quizzes, multimedia learning resources, and mobile-responsive design to bridge theoretical knowledge and practical healthcare training
# Features

Responsive web interface

AR healthcare learning content

Interactive quiz module

Embedded surgical videos

FAQ + AI-based support

Resource hub

Community forum design

Mobile optimization
